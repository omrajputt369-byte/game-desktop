### Prerequisites

### Run Locally

To run **GameX** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/omrajputt369-byte/game-desktop.git
```

Windows:

```bash
git clone https://github.com/omrajputt369-byte/game-desktop.git
```
